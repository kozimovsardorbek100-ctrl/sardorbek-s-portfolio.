Python

Python — bu 1991-yilda Guido van Rossum tomonidan yaratilgan, sodda va kuchli dasturlash tili.U dasturchilarga tez va oson kod yozishga yordam beradi.Sun’iy intellekt, veb dasturlash, botlar, ma’lumotlar tahlili va o‘yinlar uchun ishlatiladi.

HTML

HTML (HyperText Markup Language) — bu veb-sahifalar tuzish tili. U saytning tuzilmasini (skeletini) yaratadi.Matn, rasmlar, videolar va havolalarni joylashtirasiz.Brauzerga “nima va qanday ko‘rsatish”ni belgilaysiz.

CSS

CSS (Cascading Style Sheets) — bu HTML sahifalarga dizayn va bezak beruvchi til.Ya’ni, HTML saytning tuzilmasini, CSS esa uning ko‘rinishini belgilaydi.
